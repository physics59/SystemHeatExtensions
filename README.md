# System-Heat-Extensions
System Heat Extensions is a mod that extends the System Heat environment to include thermoelectrics, thermal nozzles, and more! I do not plan on adding parts for now, and this is very much in beta.

Built for Kerbal Space Program 1.12.5 using the SystemHeat framework. Thanks to Nertea and the KSP Mod Stewards for the SystemHeat projects that provide the thermal infrastructure this mod integrates with. Also, thanks to ChatGPT for helping me navigate some environment issues and to review my code.

## Current Features

- Thermoelectric Generator
- ElectricCharge generation from temperature gradients
- Configurable efficiency
- Designed for SystemHeat integration

## Planned Features

- Additional generators
- Advanced heat engine components

## Dependencies

- ModuleManager
- SystemHeat

## C# building

1. Copy the required KSP assemblies into the `Libraries/` folder.
2. Build using .NET Framework 4.8.1
